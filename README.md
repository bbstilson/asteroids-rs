# Weekend Asteroids

Quick implementation of Asteroids built over a weekend using [Macroquad](https://github.com/not-fl3/macroquad).

## Release WASM Build

```bash
./build_wasm.sh
```

## Testing WASM Build

After building the WASM bundle, one way to host it is like so:

```bash
cargo install basic-http-server
basic-http-server .
```
